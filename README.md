# Business Computing Portfolio (30%)

## Problems Solved

### Problem 1: Cars Dataset Analysis
- Loaded car dataset, descriptive statistics, median calculation
- Scatter plot (Mileage vs Price) with analysis
- Correlation analysis (Mileage/Price, Engine_Size/Price)

### Problem 2: Amazon Food Reviews
- Loaded Amazon food review dataset
- Created word_count column
- Classified reviews as short/medium/long using for loop, while loop, and vectorisation
- Pie chart of length categories

### Problem 3: College Majors Analysis
- Loaded college majors dataset
- Correlation between median salary and unemployment rate (scatter plot)
- Filtered majors with high salary (£60k+) and low unemployment (<5%)
- Grouped by Major_category for average median salary
- Bar chart visualisation

### Problem 4: Hospital Database (SQLite)
- Created Hospital_System.db with Hospital and Wards tables
- Inserted provided data
- Displayed wards with hospital locations and hospitals without wards
